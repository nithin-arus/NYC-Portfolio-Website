# nithinaruswamy.com

My personal portfolio. Live at **[nithinaruswamy.com](https://nithinaruswamy.com)**.

I'm Nithin Aruswamy. This is where I keep my work, writing, awards, and a few things I like.

## About the design

The whole site is a love letter to Massimo Vignelli — the typography, the grid, the color palette, and the navigation all borrow from his 1972 NYC Subway map and the Unimark signage system. Helvetica everywhere. Black, white, and the six MTA line colors. The sidebar pretends to be a subway platform sign; the section headers pretend to be route bullets.

## Stack

Static site. HTML, CSS, and React (no build step, no framework, no tracking). Hosted on Netlify.
